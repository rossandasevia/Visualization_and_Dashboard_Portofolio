# Dashboard

## Identifikasi Harga Konsumen Perdesaan Kab. Kediri 2025
- **Data**: Consumer price survey data in 10 traditional markets in Kabupaten Kediri (primary data)
- **Software/Tools**: Microsoft Excel, PowerBI
- **Description**: This project focuses on identifying and analyzing rural consumer prices in Kabupaten Kediri 2025. The data pre-processing includes data cleaning, handling missing values, and classifying commodities into groups of category.

## Harga Kendaraan Operasional Jawa Timur 2025 (Beli)
- **Data**: New and used vehicle purchase prices survey (primary data)
- **Software/Tools**: Microsoft Excel, PowerBI
- **Description**: This project focuses on analyzing and identifying prices of operational vehicles in Jawa Timur, covering both new and used vehicles. The data pre-processing includes data cleaning and validation, classifying each vehicles into groups of category, and identification of unusually high/low prices.
